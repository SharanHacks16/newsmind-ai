# Dataset Setup

Place the official BBC News Dataset CSV in this folder as `bbc_news.csv`, `bbc-news-data.csv`, or `BBC News Train.csv`.

The CSV must contain a text column named `text`, `article`, or `content`, and a label column named `category` or `label`.

The included `bbc_news_sample.csv` lets the project run immediately. Replace it with the full BBC News Dataset for final report metrics.
